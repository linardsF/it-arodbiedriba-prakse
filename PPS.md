# Programmas Prasību Specifikācija (PPS)

## Programmas mērķis
Šis projekts ir vienkārša web lietotne, kas demonstrē integrāciju starp GitHub un Supabase. Tā nolasa uzdevumu sarakstu no Supabase datubāzes un attēlo to lietotājam. Projekts pastāv, lai praktikanti apgūtu pamata IT prasmes, piemēram, versiju kontroli, datubāžu integrāciju un dokumentāciju. Tas ir piemērs reāliem IT projektiem, kur tiek izmantoti mākoņpakalpojumi un atvērtā koda rīki.

## Galvenie lietotāji
- Praktikanti IT jomā, kuri mācās izveidot un dokumentēt projektus.
- IT arodbiedrības biedri, kuri var izmantot to kā piemēru saviem projektiem.
- Izglītības iestāžu studenti, kuri apgūst web izstrādi.

## Ko lietotājs var izdarīt
Lietotājs var atvērt web lapu un apskatīt uzdevumu sarakstu no datubāzes. Šobrīd lietotne tikai nolasa un attēlo datus – lietotājs nevar pievienot, rediģēt vai dzēst ierakstus. Nākotnē varētu pievienot funkcionalitāti, piemēram, uzdevumu pievienošanu vai autentifikāciju.
