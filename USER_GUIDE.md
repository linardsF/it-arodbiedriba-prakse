# Lietotāja Rokasgrāmata

## Kā atvērt projektu
1. Ieej GitHub Pages adresē: https://linardsf.github.io/it-arodbiedriba-prakse/.
2. Lapa automātiski ielādēsies un mēģinās pieslēgties Supabase, lai nolasītu datus.

## Ko lietotājs redz ekrānā un ko var darīt
- Vienkāršu sarakstu ar ierakstiem, kas atrodas Supabase datubāzē (piemēram, Members tabula).
- Reālā laikā nolasītos datus, piemēram:
  - ID
  - Nosaukums
  - Apraksts vai statuss


## Ko darīt, ja kaut kas nestrādā
- Pārlādē lapu (F5 vai Ctrl+R).
- Pārbaudi interneta savienojumu.
- Ja dati neparādās, pārbaudi pārlūka konsoli (F12 > Console) uz kļūdām. Ja ir problēmas ar Supabase, pārliecinies, ka atslēgas ir pareizas.
- Ja dati neielādējas, iespējams Supabase servisam ir īslaicīgs pārtraukums.
- Ja lapa vispār neielādējas, pārbaudīt vai GitHub Pages repozitorija iestatījumi ir “published”.
